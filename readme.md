# FOLIO Voucher Export

An Interface for running simple sql queries without an IDE

## Requirements


* Python 3.x
* json
* tkinter
* sys
* psycopg2



## Instructions

* Create json config file in the following format:
>{\
        "dbname": "",\
    "user": "",\
    "host": "",\
    "password": "",\
    "query_file": ""\
}
* query_file should be the filepath for the .sql file you would like to run.
